# CSS Santa Art
CSS art of Santa Claus. This is based on Alvaro Montoro's tutorial from a CodeNewbie article.

## Tech Stack
- HTML
- CSS

## Instructions
Clone this repo or download the zip file. Open the index.html file in your browswer. You will see Santa standing in the snow with snowflakes on the background.
